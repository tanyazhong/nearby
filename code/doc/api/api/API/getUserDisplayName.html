Gets theuser display name given a user id.

We take as parameter a user's Spotify ID, and return a string representing a user's display name. This function also adheres to the information hiding principle because changes in our application needs will be isolated to this function. For instance, if we want userIDs to be associated to an app other than Spotify, we can use a different external API (instead of Spotify) and still return a String representing the user's name. pic.

Implementation

Future<String> getUserDisplayName(String userID) async {
  UserPublic user = await spotify.users.get(userID);
  return user.displayName!;
}
